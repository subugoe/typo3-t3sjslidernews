HTML-Template for contentStyle tt_content

Common markers:
NUMBER - the number of the content item (start with 1) 
IMAGE - content item image or default image
THUMBNAIL - content item thumbnail or default thumbnail
TITLE - content item header
TITLELINK - link content item header to the page with the content item
SUBTITLE - content item subheader 
DATE - date and/or time in content item
TEXT -  show the bodytext (if any)
READMORE - link a text (read more) to the page with the content item
PAGINATION




Own markers:
OWN_MARKER_ONE - specify in the EXT-Manager
OWN_MARKER_TWO - specify in the EXT-Manager




DAM markers (if using dam):
DAM_TITLE - title from the dam-record 
DAM_DESCRIPTION - description from the dam-record 
DAM_CAPTION - caption from the dam-record 
DAM_ABSTRACT - abstract from the dam-record 
DAM_CREATOR - creator from the dam-record 
DAM_PUBLISHER - publisher from the dam-record 
DAM_COPYRIGHT - copyright from the dam-record
